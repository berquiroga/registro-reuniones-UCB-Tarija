# Sistema de Registro de Tutorías y Atención a Estudiantes

Una aplicación web ligera y eficiente desarrollada en Python utilizando Streamlit. Diseñada para digitalizar y automatizar el registro de asistencia a horarios de atención, tutorías y relatorías.

**Desarrollado para:** Departamento de Tecnología e Innovación - Ingeniería Mecatrónica  
**Institución:** Universidad Católica Boliviana "San Pablo" - Sede Tarija  
**Docente:** M.Sc. Ing. Bernardo Quiroga Turdera  

---

## Propósito del Proyecto

Este sistema reemplaza los tradicionales formularios físicos en papel por una interfaz digital de "fricción cero". Permite a los estudiantes registrar su asistencia mediante el escaneo de un código QR en la puerta de la oficina, optimizando el tiempo y generando reportes estructurados automáticamente para la gestión administrativa mensual.

## Características Principales

- **Interfaz Intuitiva:** Interfaz limpia y rápida orientada al usuario final (estudiantes).
- **Clasificación de Atención:** Categorización entre Atención Regular, Tutoría de Grado, Relatoría y Tutoría de Pasantía.
- **Exportación de Datos:** Consolidación automática en formato `.csv` y descarga directa desde un entorno seguro para el administrador.
- **Despliegue en la Nube:** Preparado para su alojamiento en Streamlit Community Cloud.
